
Синтаксис


Классы и функции
    1) В пайтоне функции могут существовать отделно от класса
    2) Функция внутри класса - метод
    3) Внутри каждого метода должен быть параметр
        параметром выступает объект внутри которого вызывается метод
        напр: "def setUp(self)" где self - объект
    4) в тестовом фреймворке unittest тестовый класс должен быть наследником специального класса "unittest.TestCase"
        тестовые методы (функции) должны быть размещены внутри таких классов
